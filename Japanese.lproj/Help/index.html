DropLHa について

DropLHa は簡単に LHA アーカイブを作成するアプリケーションです。LHA は日本国内の Windows PC ユーザの間で古くからよく使われていたアーカイブ形式です。

DropLHa でアーカイブを作成するもっとも簡単な方法は、圧縮したいファイルやフォルダを DropLHa のアイコンにドラッグアンドドロップすることです。ドラッグされたファイルやフォルダは圧縮されて LHA アーカイブに格納されます。アーカイブはデフォルトではデスクトップに作成されます。例えば「書類」フォルダをドラッグすると、デスクトップに「書類.lzh」という名前のファイルができます。

DropLHa の使い方

DropLHa のアイコンに圧縮したいファイルやフォルダをドラッグします。

  1. Finder でアーカイブに格納したいファイルやフォルダを選択します。
  2. それらを DropLHa のアイコンが反転するまでドラッグします。
  3. マウスのボタンを離します。DropLHa はそれらのアイテムをアーカイブに格納し、完了すると自動的に終了します。

メニューからアイテムを選択してアーカイブを作成するには：

  1. DropLHa アイコンをダブルクリックして、DropLHa を起動します。
  2. ファイルメニューから「開く...」を選びます。
  3. ファイル選択ダイアログが表示されますのでアーカイブにしまいたいファイルを選びます。（Shift や Command キーを押しながらクリックすると複数のアイテムを選択できます）
  4. 「選択」をクリックします。

環境設定

アーカイブの保存先や形式、DropLHa の動作などを設定することができます。

保存先

デスクトップ
デスクトップにつくられます。これはデフォルトの設定です。
オリジナルと同じ
圧縮しようとしたファイルと同じフォルダにつくられます。
その都度指定
圧縮しようとするたびにアーカイブをつくるフォルダを訊ねます。
指定：
ここで指定したフォルダにつくられます。

オプション

圧縮方式：
LHA アーカイブの圧縮方式を指定します。デフォルトは lh5 です。
注意： LHA アーカイブを扱うユーティリティの中には lh6, lh7 をサポートしていないものがあります。
ヘッダレベル：
LHA アーカイブのヘッダレベルを指定します。デフォルトは 2 です。
隠しファイルを含める
このオプションが ON のとき、.DS_Store などの隠しファイルもアーカイブに含めます。通常これらのファイルは Windows では不要ですが、Mac では必要な場合があります。
アイテムごとに別々のアーカイブを作成する
このオプションが ON のとき、複数のファイルやフォルダをドラッグしたときにそれらを別々のアーカイブに格納します。
ドラッグアンドドロップで起動後、自動終了しない
このオプションが ON の場合、DropLHa はファイルをドラッグアンドドロップして起動した際、アーカイブ作成完了後に自動的に終了しなくなります。
